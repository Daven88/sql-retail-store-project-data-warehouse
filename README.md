# Independent Data Warehouse and Analytics Project
Independent data warehouse project applying industry-standard ETL workflows and dimensional modeling techniques to a retail dataset sourced from Kaggle. The project follows a Bronze–Silver–Gold architecture and integrates multiple data sources to support analytical use cases.
